# IGNITE Presentation Website 🌐🔥

This repository contains the web presentation and product showcase landing page for **IGNITE (ThermoAI Vision)**, a high-security medical imaging suite developed for the German Youth Science Competition (**Jugend forscht 2026**).

The website serves as a landing page for recruiters, judges, and users, detailing the software's architecture, its medical background in joint inflammation detection, security mechanisms, and user guides.

---

## 📂 Project Structure

The project code is organized as follows:

* **/website**: Contains a complete, modern React application powered by Vite.
  * `index.html` - Primary HTML template.
  * `vite.config.js` - Configuration for the Vite bundler.
  * `package.json` - Node.js dependencies, scripts, and configuration.
  * `src/` - React components, stylesheets, and asset managers.

---

## 🛠️ Tech Stack

* **Framework:** React
* **Build Tool:** Vite
* **Styling:** CSS3 / Tailwind CSS
* **Code Quality:** OxLint (`.oxlintrc.json` configured for high-performance static analysis)
* **Language:** JavaScript (ES6+)

## 💻 Getting Started

To launch the website preview locally:

### 1. Clone the repository
```bash
git clone https://github.com/noackjona-hash/IgniteWebseite.git
cd IgniteWebseite/website
```

### 2. Install dependencies
```bash
npm install
```

### 3. Start the Vite development server
```bash
npm run dev
```
Open the local URL displayed in the terminal to view the website.
